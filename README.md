# PacMan 🍬

A two-player PacMan game built with vanilla JavaScript and HTML Canvas.

## Play Online
[kavbian.github.io/PacMan](https://kavbian.github.io/PacMan/)

## Features
- Two-player local multiplayer
- Custom candy cane collectibles
- Enemy ghosts with random movement
- Score tracking for both players

## Controls
**Player 1:** Arrow keys  
**Player 2:** WASD

## Technologies
- Vanilla JavaScript
- HTML Canvas API
- CSS
